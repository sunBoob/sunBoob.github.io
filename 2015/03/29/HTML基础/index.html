HTML基础

一、了解HTML

​ HTML不是一个单词，而是（Hyper Text Markup Language）首字母的缩写。意思是超文本标记语言。我们来逐字翻译一下

  • 超文本：简单在字面上的意思来看就是，要比普通文字还要牛掰的语言。就想普通人和朝人一 样。
  • 标记：也被称为“标签”，超文本之所以比普通文本牛，就是因为有特殊的标记。就像超人身上的S字母一样

  • 语言：沟通方式，和浏览器沟通的语言。

    ​所以总结一下 HTML 就是通过一些带有特殊标记的语言告诉网页（浏览器）中应当显示什么，主要作用就是用来描述网页的结构和内容。相当于描述一个人 的身体结构，五脏六腑。

什么是DTD

​ 在HTML基本结构中的第一句代码是以<!DOCTYPE…>开头的标签，这个标签被称为DTD

​ DTD（Document Type Declaration）意思是文档类型声明

​ 作用就是告诉浏览器在解读网页中结构和内容时，按照哪一个HTML版本标准来解读。

二、主流HTML的版本

  • HTML4.01：

    HTML4.01的DTD有三种模式，分别是

    1. strict：严格模式

      严格模式中不推荐使用一些仅仅只是表现样式的而没有语义的标签如 b 标签、i 标签、s 标签、u 标签等，这些标签在严格模式中视为不规范

    2. Transitional：过渡模式

      过渡模式中，使用一些仅仅表现样式而没有语义的标签，也会被视为规范

    3. Frameset：框架模式

      在页面中若使用frameset标签，推荐使用该模式

  • XHTML1.0：

    XHTML1.0的DTD有三种模式，分别是

    1. strict：严格模式

      严格模式中不推荐使用一些仅仅只是表现样式的而没有语义的标签如 b 标签、i 标签、s 标签、u 标签等，这些标签在严格模式中视为不规范

    2. Transitional：过渡模式

      过渡模式中，使用一些仅仅表现样式而没有语义的标签，也会被视为规范

    3. Frameset：框架模式

      在页面中若使用frameset标签，推荐使用该模式

  • HTML5.0：

    1. HTML5.0是最新版本，与之前版本不同的是，增加了很多新的特性：

      以往用flash插件来实现媒体功能，比如：视频、音频、画布等

  • XHTML1.0 和 HTML4.01的区别

    1. XHTML1.0比HTML4.01在语法上规范要求更加严格

      如：双标签要承兑、html以及其中的标签要小写

      区别是理论上的区别，表现上无法区别

3、图片 img标签

  • 写法 :
  • 作用 : 在页面中插入一张图片。

  • 属性 : src=”” 表示图片的路径，就是在哪里找到这张图片

    ​ alt=””表示图片在加载异常的时候用文字代替图片表达的意思。

    ​ title=””表示图片的标题，表示当鼠标悬停在图片上的时候显示图片信息

    ​ width=””表示图片的宽度。

    ​ height=””表示图片的高度。

    ​图片的宽度高度，单独设置的时候，图片会根据，width 和 height 所给的值，进行等比例缩放。在实际开发中不会使用这种方式表示。会使用css样式来设置图片的宽高。

  • 路径的问题 :

    • 相对路径:

      1. 情况一：图片和HTML文件在同一个文件夹里的时候，可以直接输入图片的 图片名.图片格式即可 src=”sunbobo.jpg”
      2. 情况二：图片在HTML文件的上层文件夹的时候，HTML文件若想访问这张图片需要出去这层文件夹，才能找到图片。src = “../sunbobo.jpg” ../的意思就表示出去本层目录（文件夹）
      3. 情况三：图片在HTML文件的下层文件夹的时候，HTML文件若想访问这张图片需要进去图片所在层文件夹，才能找到图片。比如说图片在 images文件夹中的时候。HTML想要访问这张图片src = “images/sunbobo.jpg” 表示进去images文件夹访问这张图片
      4. 情况四：图片HTML问价在不同的文件夹中的时候，如 图片在images文件夹中，HTML文件，在HTML文件夹中。HTML文件若想访问这张图片需要出去HTML文件夹再进入images文件夹中，才能找到图片。src = “../images/sunbobo.jpg” ..表示先出去本层HTML文件夹，然后进入images文件夹访问图片。

      ​ 这种访问路径的方式叫做相对路径。

      ​路径有两种表示方式，一个是相对路径，一个是绝对路径。

    • 绝对路径：不常用（简单了解）

      1. 就是一堆带有盘符的路径访问方式叫做绝对路径。比如src=”C:\Users\bhjt\Desktop\images\sunbobo.jpg”

      2. 或是带有网址的路径也被称为绝对路径。比如

        src=”www.baidu.com”
